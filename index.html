Draw Church — метрические книги Российской Империи

Интерактивный граф родственных связей из метрических книг Российской Империи (XIX век).

Открыть граф GitHub

Граф работает в любом современном браузере. Для больших компонент может потребоваться несколько секунд на загрузку.

Раскраска узлов

Каждое поселение имеет свой цвет. Цвет узла = цвет поселения, к которому приписана персона.

Размер узла пропорционален количеству связей: чем больше рёбер, тем крупнее точка.

Компоненты связности

Граф разбит на компоненты связности — персоны внутри одной компоненты связаны между собой (через цепочки рождений, браков, восприемничества). Выпадающий список в тулбаре переключает между компонентами.
