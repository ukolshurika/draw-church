# Draw Church

Интерактивный граф родственных и социальных связей из метрических книг
Российской Империи (XIX век), размещённых в [Яндекс.Архиве](
https://yandex.ru/archive).

**1772 персоны, 2245 связей** — рождения, браки, восприемники, свидетели —
из Богоявленского прихода г. Серпухова, 1882 г.

![graph example](graph.html)

## Pipeline

```
links.md  ──►  scraper.py  ──►  raw_api/  ──►  parse.py  ──►  all-nodes.json
                                                               all-edges.json
                                                                  │
                                                     normalize_geo.py ──►  viz.py  ──►  graph.html
```

| Шаг | Скрипт | Назначение |
|-----|--------|------------|
| 1 | `scraper.py` | Скачать данные из Яндекс.Архива через Playwright |
| 2 | `parse.py` | Извлечь персон и связи, дедуплицировать |
| 3 | `normalize_geo.py` | Нормализовать названия поселений |
| 4 | `viz.py` | Построить `graph.html` (vis-network) |

## Запуск

```bash
pip install playwright networkx
playwright install chromium

python3 scraper.py          # скачать (resume-ready)
python3 parse.py            # извлечь + дедуплицировать
python3 normalize_geo.py    # нормализовать названия
python3 viz.py              # построить graph.html
```

Открыть `graph.html` в браузере.

## Возможности визуализации

- **59 компонент связности**, главный — 1575 персон
- Раскраска по поселениям (32 цвета)
- Размер узла пропорционален количеству связей
- Поиск по имени / фамилии / поселению
- Фильтры: только рождения, только браки, все
- Боковая панель: легенда, детали персоны по клику
- **Двойной клик** — режим фокуса: граф сужается до персоны и соседей в радиусе N рёбер (слайдер 1–10 в боковой панели, по умолчанию 3; Esc / × — выход)
- Адаптивная физика под размер компоненты
- **Кнопка ⚡ / P** — вкл/выкл физику (по умолчанию включена, граф «дышит»)
- **Кнопка 💾** — сохранить текущий вид как PNG

## Типы связей

| Связь | Цвет | Описание |
|-------|------|----------|
| `child_of` | Красный | Ребёнок → родитель |
| `godparent_of` | Оранжевый | Восприемник → ребёнок |
| `married_to` | Розовый | Жених ↔ невеста |
| `witnessed_for` | Серый | Свидетель брака |

## Область данных

Богоявленская церковь г. Серпухова, 1882 г. Охватывает Пущинскую,
Высотскую, Туровскую, Кошкинскую волости Серпуховского, Алексинского,
Тарусского, Каширского, Одоевского и смежных уездов.

Скрипты не привязаны к приходу — можно использовать для любых метрических
книг Яндекс.Архива. При смене прихода необходимо расширить словарь
`settlement-synonyms.json` (см. `normalize_geo.py`).
