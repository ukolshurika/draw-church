# Draw Church — метрические книги в граф связей

**Draw Church** превращает записи из метрических книг (рождения, браки) Российской Империи XIX века в интерактивный граф, где видно, кто кому кем приходится: родители, дети, крёстные, супруги, свидетели на свадьбе.

Данные взяты из [Яндекс.Архива](https://yandex.ru/archive)

## Что нужно установить
 - скачать `opencode`.
 - нати на яндекс метриках интересующую книгу и скопировать url с номерами страниц. Например: "https://yandex.ru/archive/catalog/00f0a2cd-2e84-46cc-bfd4-8bc9b9c838b0?sheet_page_from=109&sheet_page_to=163"
 - запустить opencode из папки с проектом и написать промт вида:

 "построй граф для url https://yandex.ru/archive/catalog/00f0a2cd-2e84-46cc-bfd4-8bc9b9c838b0?sheet_page_from=109&sheet_page_to=163, https://yandex.ru/archive/catalog/00f0a2cd-2e84-46cc-bfd4-8bc9b9c838b0?sheet_page_from=109&sheet_page_to=163, назови выходные файды с суффиксом alexeevskaya_1900_1905"

 Посде ПРОДОЛЖИТЕЛЬНОЙ работы у вас в папаке появяится html файл, который открывается двойным кликом в вашем браузере по умолчанию.


## Как слить персон, которые не соединились автоматом:
В `opencode` написать промт вида:

```
У Михаил Алексеев Рыжев (1882) есть много связей married.
На самом деле их было две: Марфа Егорова (она же Марфа Егорова Ершова)
и Елена Егорова Рыжева (она же Елена Георгиева, Елена Егорова Рыдева
из села Хотушь). Объедини.
```


## Если видите справа в населенных пунктах какие-то не корректно распознанные поселения

Пишем в `opencode`:

```
Село X и село Y на самом деле одно и тоже. используй /geo-normalization и объедини их
```
